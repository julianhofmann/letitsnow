=============
Configuration
=============

"Let it snow!" can be used in two different (mutually exclusive) ways:

Global
======

The extension ships some TypoScript code which needs to be included for
*global* usage.

#. Switch to the root page of your site.

#. Switch to :guilabel:`Template > Info/Modify`.

#. Press the link :guilabel:`Edit the whole template record` and switch to the
   tab :guilabel:`Includes`.

#. Select :guilabel:`Let it snow: Global (letitsnow)` at the field :guilabel:`Include static (from extensions):`

   .. image:: /Images/LetitsnowIncludeTypoScriptGlobal.png


Plugin
======

The extension ships some other TypoScript code which needs to be included for
*plugin-based* usage.

#. Switch to the root page of your site.

#. Switch to :guilabel:`Template > Info/Modify`.

#. Press the link :guilabel:`Edit the whole template record` and switch to the
   tab :guilabel:`Includes`.

#. Select :guilabel:`Let it snow: Plugin (letitsnow)` at the field :guilabel:`Include static (from extensions):`

   .. image:: /Images/LetitsnowIncludeTypoScriptPlugin.png
